Now Link for Drupal 7.x
----------------------------
The Now Link module provides a link for date fielws that will update thier values to the current timestamp.
The drupal admin can select which node type and date field to assign the link to, as well as which view mode
to display the link on.

Installation
------------
The Now Link module can be installed like any other Drupal module -- place it
in the modules directory for your site and enable it on the 'admin/build/modules'
page.
